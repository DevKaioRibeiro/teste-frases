API de Filmes

API RESTful desenvolvida em Node.js com Express e MongoDB. Fornece endpoints para buscar, criar, atualizar e remover informações de filmes e avaliações.

Tecnologias: Node.js, Express, MongoDB, JWT para autenticação.

Ideal para consumo por aplicações web ou mobile, a API foi documentada usando Swagger/OpenAPI e possui testes automatizados.
